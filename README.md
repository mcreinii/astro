#### One liner command (If you don't have any other configuration)
```shell
mv ~/.config/nvim ~/.config/nvim.bak && mv ~/.local/share/nvim ~/.local/share/nvim.bak && mv ~/.local/state/nvim ~/.local/state/nvim.bak && mv ~/.cache/nvim ~/.cache/nvim.bak && git clone --depth 1 https://github.com/mcreinii/astro ~/.config/nvim
```

#### Make a backup of your current nvim and shared folder

```shell
mv ~/.config/nvim ~/.config/nvim.bak
mv ~/.local/share/nvim ~/.local/share/nvim.bak
mv ~/.local/state/nvim ~/.local/state/nvim.bak
mv ~/.cache/nvim ~/.cache/nvim.bak
```

#### Install this configuration

```shell
git clone --depth 1 https://github.com/mcreinii/astro ~/.config/nvim
```
